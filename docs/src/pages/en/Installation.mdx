## Prerequisites

- node _1.16_ or higher
- terminal - for installing HuggerEngine you need to use a terminal
- Text editor - We recommend VS Code

## automatic install

install HuggerEngine automatic with our cli. just run the following command in your terminal

```bash
npx create-HuggerEngine
```

You don't need to create a new directory, our cli will create one for your project.
